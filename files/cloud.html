📌 Token 创建步骤：
1️⃣ 点击"创建 Token"按钮（或直接访问：https://github.com/settings/tokens/new）
2️⃣ 在 "Note" 栏填写备注（如：Voice Assistant Cloud）
3️⃣ 向下滚动到 "Select scopes" 区域
4️⃣ 勾选 "repo" 权限（最重要！）
5️⃣ 滚动到页面底部，点击绿色 "Generate token" 按钮
6️⃣ 复制生成的 Token（以 ghp_ 开头），粘贴到上方输入框

⚠️ 重要提示：
• Token 只会显示一次，请立即复制保存
• Token 仅保存在本地浏览器，不会上传到任何服务器
• 可随时在 Token 管理页面 撤销

📖 使用教程

☁️ GitHub 云盘使用说明：

1️⃣ 获取 GitHub Token（点击"获取 Token"按钮）

2️⃣ 创建 Personal Access Token，勾选 repo 权限

3️⃣ 保存 Token 后，上传文件到 GitHub 仓库

4️⃣ 在任何设备登录同一 GitHub 账号即可访问文件

💡 优势：文件存储在 GitHub，永久免费，跨设备同步，无需担心丢失。
